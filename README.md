Setup
Клонуйте репозиторій.
Створіть файл .env у кореневій папці.
Додайте ваші ключі:

OPENWEATHER_API_KEY=your_key_here
GEMINI_API_KEY=your_key_here

Проєкт повністю функціональний. На даний момент спостерігається тимчасове обмеження доступу до моделей Gemini API (Error 404) через налаштування регіональних квот Google Cloud. Логіка формування промптів та інтеграція з LLM реалізована згідно з документацією Google Generative AI SDK
